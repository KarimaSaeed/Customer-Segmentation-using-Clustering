# Customer-Segmentation-using-Clustering
This project applies unsupervised machine learning techniques to segment customers into distinct groups based on their behavior and characteristics.
The goal is to identify patterns in customer data to support targeted marketing and business decision-making.

---

## 🔍 Exploratory Data Analysis (EDA)
- Performed data exploration and visualization
- Identified patterns and relationships between features
- Prepared data for clustering

---

## 🧹 Data Preprocessing
- Handled missing values
- Applied feature scaling
- Selected relevant features for clustering

---

## 🤖 Model Development
- Implemented **K-Means clustering**
- Determined optimal number of clusters using:
  - Elbow Method
  - Silhouette Score

---

## 📈 Results
- Successfully segmented customers into distinct groups
- Visualized clusters using Matplotlib and Seaborn
- Extracted insights to support business strategies

---

## 💡 Key Insights
- Customers can be grouped based on behavior and characteristics  
- Clustering helps improve targeting and personalization strategies  

---

## 🛠️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
